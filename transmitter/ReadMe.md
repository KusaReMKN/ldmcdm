# 送信機

シリアルデバイスから入力されたデータを受信機に向けて送信します。

## つかいかた

Linux や macOS では**[送信機用ドライバプログラム]**を使うと便利です。
このプログラムは、送信機と適切に通信することにより、必要な操作を自動的に行います。

手動で制御を行う場合は、適当な端末ソフトウェアをつかって接続してください。

まず、送信速度を設定する必要があります。
送信機に **CR**（復帰文字）を送信すると `?` を応答します。
これは、送信速度を設定するモードであることを示しています。
送信速度（送信機におけるクロック速度）を 10 進数で送信し、**CR** を送信します。
送信機はプレアンブルの送信を開始します。

送信速度が設定されると、送信速度を 10 進数で応答し、続けて`!` を応答します。
これは、データを送信できるモードであることを示しています。
送信したいデータを送信します。
送信のバッファが溢れないよう、小分けにして送信する必要があるかもしれません。

送信機は送信すべきデータを送信しおえると、`?` を応答して動作を停止します。

つまり、次のように送信すると、速度 300 で hoge を送信できます。

```
300
hoge
```

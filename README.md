# レベル分割多重と符号分割多重とを用いた簡易可視光通信システム

2025 年 11 月 28 日から 29 日にかけて、
東京都小平市で開催された「PTU フォーラム 2025」（第 33 回 職業能力開発研究発表講演会）で発表された
「レベル分割多重と符号分割多重とを用いた簡易可視光通信システム」の関連プログラムです。

## 発表資料

<https://kusaremkn.com/slides/> で公開されています。

## ファイル一覧

- driver:
  Linux の上で動作する、復号ビット誤り率を測定するためのプログラムです。

- receiver:
  受信機のプログラムです。
  PlatformIO が必要です。

- tdriver:
  Linux や macOS の上で動作する、送信機を簡便に制御するためのプログラムです。

- transmitter:
  送信機のプログラムです。
  PlatformIO が必要です。
